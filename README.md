# J2MEGameCenter
An Application for Looking J2ME Applications

## Requirements

+ Create three Screen.
+ Create Room Percistent DB.
+ Uses Shared Preferences .
+ Add Preferences Activity.
+ Add API REST Services.
+ Uses JobScheduler Library.
+ Add Favorite Activity.
+ Add Details Activity.
